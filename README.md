# SwiftlyChat

**SwiftlyChat** is an iOS messaging app built with **Swift** and **Firebase**, designed to demonstrate real-time communication, user authentication, and cloud data management.

<br/>

## Features

- **Firebase Authentication** – Create accounts and log in with email, password, and a profile picture.  
- **User Management** – Store user data (**UID**, **email**, and **profile image URL**) in **Firestore**, and manage profile images in **Firebase Storage**.  
- **Real-Time Messaging** – Send and receive text and image messages instantly between users.  
- **User Directory** – Start new conversations from a dynamically loaded list of users stored in Firestore.  
- **Recent Chats** – View a list of recent conversations with message previews.  
- **Chat Interface** – Display the recipient’s name, profile picture, and chat history within each conversation.  
- **Secure Logout** – Sign out safely using Firebase Authentication.  

<br/>

## Tech Stack

- **Language:** Swift  
- **Backend:** Firebase (**Authentication**, **Firestore**, **Storage**)  
- **Platform:** iOS  

<br/>

## Summary

**SwiftlyChat** integrates multiple Firebase services to provide a seamless, real-time chat experience.  
It’s built as a practical demonstration of **authentication flows**, **database synchronization**, and **media handling** in modern iOS app development.
